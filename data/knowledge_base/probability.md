# Probability Knowledge Base

## Basic Probability

P(E) = favorable outcomes / total outcomes

0 ≤ P(E) ≤ 1

## Complement Rule

P(A') = 1 − P(A)

## Addition Rule

P(A ∪ B) = P(A) + P(B) − P(A ∩ B)

## Multiplication Rule

P(A ∩ B) = P(A) * P(B | A)

If independent:

P(A ∩ B) = P(A) * P(B)

## Conditional Probability

P(A | B) = P(A ∩ B) / P(B)

## Bayes Theorem

P(A | B) = [P(B | A) P(A)] / P(B)
