# Arithmetic Knowledge Base

## Basic Operations

Addition: a + b  
Subtraction: a - b  
Multiplication: a * b  
Division: a / b

## Percentage

Percentage = (part / whole) * 100

## Profit and Loss

Profit = SP − CP  
Loss = CP − SP

Profit % = (Profit / CP) * 100

Loss % = (Loss / CP) * 100

## Simple Interest

SI = (P * R * T) / 100

## Average

Average = sum of values / number of values

## Ratio

a : b = a / b
