# Statistics Knowledge Base

## Mean

Mean = (Σx) / n

## Median

Middle value of sorted data

If even count:
Median = (n/2 + n/2 + 1) / 2

## Mode

Most frequently occurring value

## Variance

Population variance:

σ² = Σ(x − μ)² / N

## Standard Deviation

σ = √variance

## Range

Range = max value − min value
