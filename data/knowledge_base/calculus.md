# Calculus Knowledge Base

## Derivative Rules

1. d/dx(x^n) = n*x^(n-1)
2. d/dx(sin x) = cos x
3. d/dx(cos x) = -sin x
4. d/dx(e^x) = e^x
5. d/dx(ln x) = 1/x

## Product Rule

d/dx[f(x)g(x)] = f'(x)g(x) + f(x)g'(x)

## Quotient Rule

d/dx[f(x)/g(x)] = (f'(x)g(x) - f(x)g'(x)) / g(x)^2

## Chain Rule

d/dx[f(g(x))] = f'(g(x)) * g'(x)

## Basic Integrals

1. ∫ x^n dx = x^(n+1)/(n+1) + C
2. ∫ 1/x dx = ln|x| + C
3. ∫ e^x dx = e^x + C
4. ∫ sin x dx = -cos x + C
5. ∫ cos x dx = sin x + C
